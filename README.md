# StudyMate-SDG-4 📚

StudyMate is a modern Android Study Planner and learning application built with **Kotlin** and **Jetpack Compose**. 

The application is designed to support **SDG 4: Quality Education** by providing students with a clean, organized, and high-performance tool to manage their academic schedules, exams, and tasks.

## ✨ Features
- **Liquid Glass UI:** A visually stunning interface using custom gradients and blurred-effect surfaces.
- **Dynamic Theming:** Fully integrated with Material 3 Design (Light & Dark modes supported).
- **Interactive Planner:** Expandable info cards with smooth spring animations.
- **Study Assistant:** A built-in search/chat interface for quick academic queries.
- **Calendar View:** A horizontal daily schedule tracker.

## 🛠 Tech Stack
- **Language:** Kotlin
- **Toolkit:** Jetpack Compose (Modern UI)
- **Design System:** Material Design 3
- **Animations:** Compose Animation API (Spring Physics)
